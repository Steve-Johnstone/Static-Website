Continous Deployment through AWS

Welcome to my static website, which is deployed using an S3 bucket and updated via github, when new commits are pushed.

Key steps in this demo:

  1. Create a new S3 bucket in AWS.
  2. Deploy a static website to the newly created bucket.
  3. Set up a pipeline for the bucket, linked to a Github account.
  4. Make changes to the website and push them to Github.
  5. The changes are then deployed to the this site.
